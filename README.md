# Mario made with [PyGame](https://www.pygame.org)

This is a simple project to NSI . To see more, get the official pygame news [here](https://www.pygame.org).



### Running

    $ git clone https://github.com/Hugo-Galley/Mario_in-Pygame
    $ cd projetMario
    $ python3 src/main.py
